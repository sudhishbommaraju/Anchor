Features

Everything, on every call.

The full toolkit for keeping AI coding agents on task — continuous memory, semantic detection, live spend, and per-mission control.

Continuous memory injection

Mission plus a deduped, living memory rebuilt and re-injected as a fresh system block on every model call.

Semantic loop & drift detection

pgvector similarity catches repeats and off-task steps, then injects a corrective to re-ground the agent.

Live monitoring

Watch tokens, cost, loops, and interventions update in real time — with the exact step where it looped.

Drop-in everywhere

Works with Claude Code, Cursor, and any OpenAI- or Anthropic-compatible agent — or read the key directly via REST/MCP.

Bring your own key

Bill calls to your own provider key with BYOK, or let Anchor route through the gateway — your choice, per mission.

Sequencing & plan mode

Optionally decompose the goal into ordered steps and feed them one at a time, so the agent advances a plan.

Why a layer beats a longer prompt.

A one-shot system prompt fades as the conversation grows. Anchor acts on every call.
